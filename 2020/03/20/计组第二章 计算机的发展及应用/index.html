2.1.2 微型计算机的出现和发展

1971年，摩尔定律：当价格不变时，集成电路上可容纳的元器件的数目，约每隔18-24个月便会增加一倍，性能也将提升一倍。换言之，每一美元所能买到的电脑性能，将每隔18-24个月翻一倍以上

2.1.3 软件技术的兴起和发展

各种语言

  • 机器语言
    • FORTRAN 科学计算和工程计算
  • 汇编语言
    • PASCAL 结构化程序设计
  • 高级语言
    • c++ 面向对象
    • java 适应网络环境

系统软件

  • 语言处理程序
  • 操作系统
  • 服务器程序
  • 数据库管理系统
  • 网络软件

软件发展特点

  1. 开发周期长
  2. 制作成本昂贵
  3. 检测软件产品质量的特殊性
  4. 开发效率低
